# Almadar Homebrew Tap

Homebrew formulae for [Almadar](https://almadar.io) tools.

## Installation

```bash
brew tap almadar-io/tap
brew install almadar
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `almadar` | Almadar CLI - Compile Orbital schemas to full-stack applications |

## Updating

```bash
brew update
brew upgrade almadar
```
